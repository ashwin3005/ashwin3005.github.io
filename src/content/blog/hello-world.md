Welcome to my corner of the internet.

I've been meaning to start a blog for a while. Not to chase an audience or build a brand, but to have a place to think out loud and keep a record of things I've figured out.

## What I'll write about

Mostly tech — systems programming, web development, tools I build and use, open source. Occasionally something less technical when it seems worth writing down.

I'll try to keep posts focused and useful. Long where the topic demands it, short where it doesn't.

## The setup

The site is built with [Astro](https://astro.build), hosted on GitHub Pages, and written entirely in Markdown. It's fast, lightweight, and easy to maintain — which means I'm more likely to actually post here.

If you find something useful, that's great. If you spot an error, feel free to reach out.

More soon.
